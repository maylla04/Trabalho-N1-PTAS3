Trabalho de PTAS3

principais linguagens:

  • JavaScript
  • HTML
  • Sql postgress

componentes:

  • Config: conecta ao banco de dados
  • Controler Contem os métodos: Update(atualizar), delete(deletar), create(criar),authentic(autenticar) , find( encontrar o usuário) e createUser (cadastra o usuário)

Rotas e métodos

  • http://localhost:3000/user : utiliza o método post -- rota responsável pelo cadastro, utilizando também o createUser
  • http://localhost:3000/users : utiliza o método get -- rota responsável por encontrar o usuário, utilizando também o FindUser
  • http://localhost:3000/user/:id : utiliza o método delete -- rota responsável por deletar o usuário, utilizando também o deleteUser
  • http://localhost:3000/user/:id utiliza o método put - rota responsável por atualizar os dados dos usuários , utilizando também updateUser
  • http://localhost:3000/user/authenticated utiliza o método post - rota responsável por autenticar os dados dos usuários (coloca-los), utilizando também authenticatedUser
